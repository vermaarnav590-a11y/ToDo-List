# 📝 To-Do List Manager

A simple command-line To-Do List Manager built using Python.

## 📌 Features

- View Tasks
- Add New Tasks
- Delete Tasks
- Easy Menu-Based Interface
- Beginner Friendly

## 🛠️ Technologies Used

- Python 3

## 📂 Project Structure

```
ToDo-List/
│
├── main.py
├── README.md
└── screenshots/
    ├── menu.png
    ├── add-task.png
    ├── delete-task.png
```

## ▶️ How to Run

1. Install Python 3
2. Open the project folder
3. Run:

```bash
python main.py
```
